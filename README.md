# Data Engineering

This repository contains **Data Engineering and Analytics Engineering projects**.

The goal of this repository is to document practical implementations of modern data engineering concepts such as distributed data processing, analytics-ready dataset modeling, and scalable data pipelines.

The projects demonstrate how raw data can be transformed into reliable and structured datasets used for analytics and business intelligence.

---

# Projects

## Customer Analytics KPI Pipeline

This project demonstrates a **distributed analytics pipeline built with PySpark**.

The objective of the pipeline is to transform raw customer transaction data into aggregated metrics that can be consumed by business intelligence dashboards.

The pipeline processes transactional datasets and generates structured KPI tables that allow business teams to analyze customer behavior and performance indicators.
